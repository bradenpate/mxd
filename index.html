How do I add my new email signature to Outlook?

For Outlook on Windows

  1. Select New Email.
  2. From the Insert section of the ribbon, select Signature > Signatures
  3. Select New, type a name for the signature, and select OK.
  4. Under Edit signature, paste your signature.
  5. With your new signature selected from the list above the editing box, go to Select default signatures and choose whether to apply the signature to new messages and/or to replies and forwards.
  6. Select New Email to see the signature you created.

For Outlook on Mac

  1. Go to Outlook > Settings > Signatures.
  2. Select the “Standard” signature and click Edit, or create a new signature.
  3. Paste from your clipboard into the signature field.
  4. Select Save.
  5. You can now set this signature as the default for new and reply messages.
